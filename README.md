# Tools

A collection of miscellaneous HTML+JavaScript tools, inspired by [tools.simonwillison.net](https://tools.simonwillison.net/).

These are simple, single-file tools that run entirely in the browser.

## Usages

Simply open any of the `.html` files in your web browser.

## Tools

- [Dark Mode PDF Reader](dark-pdf-reader.html) - Read PDFs with inverted colors for easier reading in low light. Drag and drop or open any PDF file.

## Contributing

1. Create a new `.html` file for your tool.
2. Include all necessary CSS and JavaScript within the single file.
3. Add it to this README.
